Maths Shortcuts

If two numbers be a and b and their LCM and HCF be x and y then,

  1. Basic Properties :
    1. a * b = x * y
    2. LCM (a , b) = (a * b) / HCF
    3. HCF ( a , b ) = (a * b) / LCM
  2. ( a + b ) / HCF = NUMERATOR + DENOMINATOR of ( a / b )
  3. ( a * b ) / (HCF)2 = NUMERATOR * DENOMINATOR of ( a / b )
  4. LCM / HCF = NUMERATOR * DENOMINATOR of ( a / b )
  5. If a : b : c : d is given and HCF is given then , - 1st Number is = a * HCF , 2nd Number is = b * HCF and so on.
  6. For Fractions :
    1. LCM = LCM(NUMERATORS) / HCF(DENOMINATORS)
    2. HCF = HCF(NUMERATORS) / LCM(DENOMINATORS)
  7. For two Prime numbers a , b : HCF = 1 , LCM = a * b
